🎯 Quizminia – Interactive Quiz Web Application

An engaging and user-friendly quiz platform built to test knowledge, track performance, and generate certificates. This project demonstrates full-stack development using Streamlit and database integration.

---

🚀 Project Overview

Quizminia is a web-based quiz application designed to provide an interactive learning experience. It allows users to register, attempt quizzes, view scores, and download certificates — all in a smooth and simple interface.

---

✨ Features

🔐 User Authentication

- Secure Signup & Login system
- Session management for users

🧠 Quiz System

- Multiple-choice questions
- Dynamic question handling
- Score calculation in real-time

📊 Performance Tracking

- Displays user scores
- Tracks quiz attempts

📜 Certificate Generation

- Automatically generates certificates
- Downloadable PDF format using ReportLab

🎨 User Interface

- Clean and responsive UI using Streamlit
- Centered layout with logo and headings

---

🛠️ Tech Stack

Frontend:

- Streamlit

Backend:

- Python

Database:

- MySQL

Libraries Used:

- mysql-connector-python
- reportlab
- streamlit

---

⚙️ How It Works

1. User registers or logs into the system
2. User selects and attempts a quiz
3. System evaluates answers and calculates score
4. Results are displayed instantly
5. Certificate is generated based on performance

▶️ How to Run Locally

1. Clone the repository

2. Navigate to the project folder

cd quizminia

3. Install dependencies

pip install -r requirements.txt

4. Run the application

streamlit run quiz_app.py
